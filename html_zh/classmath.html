◆ random()

static function random ( local  m,
local  n 
)

当不带参数调用时，返回一个 [0,1) 区间内一致分布的浮点伪随机数。

当以两个整数 m 与 n 调用时， math.random 返回一个 [m, n] 区间

内均匀分布的整数伪随机数。

（值 n-m 不能是负数，且必须在 Lua 整数的表示范围内。）

调用 math.random(n) 等价于 math.random(1,n)。

这个函数是对 C 提供的位随机数函数的封装。

对其统计属性不作担保。

重载
fun():number
参数
mnumber
nnumber
返回
