Detailed Description

类型 userdata 用来将任意 C 数据保存在 Lua 变量中。

这个类型相当于一块原生的内存，除了赋值和相同性判断，Lua 没有为之预定义任何操作。

然而，通过使用 metatable （元表） ，程序员可以为 userdata 自定义一组操作。

userdata 不能在 Lua 中创建出来，也不能在 Lua 中修改。这样的操作只能通过 C API。

这一点保证了宿主程序完全掌管其中的数据。
